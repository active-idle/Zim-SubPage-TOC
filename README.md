# Zim Wiki SubPage TOC Plugin
A Zim Desktop Wiki plugin that generates hierarchical table of contents for subpages at the cursor position.

![Example](./image01.png)

## Features
- Automatic TOC Generation: Creates a formatted table of contents for all subpages under the current page
- Configurable Depth: Set maximum nesting level (1-10) for subpage hierarchy
- Quick Access: Insert TOC with keyboard shortcut (Shift+Ctrl+T)
- Smart Formatting: Automatically indents nested subpages with wiki-style links

## Installation
1. Copy subpage_toc.py to your Zim plugins directory:
   - Linux: ~/.local/share/zim/plugins/
   - Windows: %APPDATA%\zim\data\zim\plugins\
   - macOS: ~/Library/Application Support/zim/plugins/
2. Restart Zim Desktop Wiki
3. Enable the plugin:
   - Go to Edit → Preferences → Plugins
   - Check SubPage TOC
   - Configure maximum depth if needed (default: 1 level)

## Requirements
- Zim Desktop Wiki 0.73 or higher
- Python 3.6+
